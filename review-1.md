1. ¿Qué es mi producto y para que sirve?
   Es un aplicación de un servidor de lista de tareas desarrollado con Express, conectado a una Base de Datos en a nube mediante la cual podrás acceder
   a los datos de tus tareas desde cualquier lugar, y así optimizarás tu tiempo.

2. ¿Cuáles son las funcionalidades más importantes y porque los usuarios las usarían?
   Esta aplicación te permitirá:
   2.1 Crear cada una de las tareas con sus campos: tarea, descripción, estado (completa o incompleta).
   2.2 Actualizar cada una de las tareas por su Id.
   2.3 Eliminar las tareas que ya hayas finalizado, o que no desees que hagan parte de tu lista de tareas.
   2.4 Listar la lista con todas las tareas, o solo listar las completadas o las incompletas.

   Deseo que esta herramienta sea de mucha utilidad y te ayude a manejar tu tiempo de manera eficiente. Adiós a la procrastinación.
